      What this page is. The instruments are the same code the batteries run and the mfg-lab Python package ships — this page is generated from them, and a gate asserts the embedded copies are byte-identical to their sources, so what your browser runs is what the tests proved. Your kernel is evaluated in this page only; nothing is uploaded.

      Want to bring a whole mean-field game rather than a solver? That is the companion page. The same two instruments exist in Python, as mfglab.lab — not published to PyPI yet, so install it from the repository rather than with pip install. A cross-language differential holds the two implementations to the same decisions, so the toggle between them is a checked claim.
