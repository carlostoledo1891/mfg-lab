What holds it together

Nothing here is asserted that a machine does not re-check. One command runs all 18 batteries in this repository, and the pages are generated from the code the batteries validate — so what a browser shows is byte-identical to what was proved, and a gate fails if it drifts.

  • Certificates over tolerances. Pre-update residuals only; an identity that is zero by construction is a fake certificate and is banned.
  • A green suite proves nothing until it can go red. Every fix is mutation-tested by reverting it and confirming the check fails.
  • Failures are published. Where a comparison flatters the baseline, the number ships anyway.
  • Two implementations, one decision. The instruments exist in JavaScript and in Python, and a cross-language differential holds them to the same verdicts.
